# Livros2B

Projeto estático para controle de empréstimos de livros.

## Publicar no GitHub Pages

1. Envie o projeto para o repositório no GitHub.
2. Vá em Settings > Pages.
3. Em Source, escolha GitHub Actions.
4. O deploy será feito automaticamente ao enviar para a branch main.

## Arquivos principais

- index.html
- styles.css
- app.js
